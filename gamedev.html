Snake Clone

Um clone do clássico jogo Snake desenvolvido em Python e Pygame como projeto de aprendizado e estudo de conceitos fundamentais de game development. O jogo implementa mecânicas tradicionais como movimentação contínua, colisão, sistema de pontuação e crescimento progressivo da cobra. A release mais recente está disponível gratuitamente para download e teste.
